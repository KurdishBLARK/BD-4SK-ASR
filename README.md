# BD-4SK-ASR
An experimental dataset of Soranin Kurdish that could be used in speech recognition using CMUSphinx. The data is under the CC BY-NC-SA 4.0 license.
To cite this work please refer to the related article entitled "Kurdish (Sorani) Speech to Text: Presenting and Experimental Dataset" by Akam Omer and Hossein Hassani on arxiv.
